# Quentin et les Carapils

Petit jeu : dirige Quentin (casquette TRUMP, salopette kaki) et attrape un maximum de canettes de Carapils en 60 secondes.

## Comment jouer

Ouvre `index.html` dans ton navigateur, ou joue en ligne.

**Commandes :**
- Fleches gauche / droite : bouger Quentin
- R : rejouer apres la fin

Ton record est sauvegarde dans ton navigateur.
